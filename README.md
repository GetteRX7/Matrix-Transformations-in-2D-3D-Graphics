# 2D & 3D Matrix Transformations (Python + Matplotlib)

This project demonstrates **2D and 3D geometric transformations** using **homogeneous transformation matrices**.  
It includes:

- Translation  
- Rotation  
- Scaling  
- Shearing  
- Reflections  
- Combined transformations  
- 2D square visualization  
- 3D cube visualization  

All visualizations are generated using **Matplotlib**.
